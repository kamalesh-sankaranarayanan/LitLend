# 📚 Literature Lending App

## Overview
The Literature Lending App is a web-based application developed to manage book lending operations efficiently. It allows users to browse books, borrow available books, return books, and manage lending records digitally.

The system simplifies library management and improves accessibility for users.

---

## Features
- User Registration & Login
- Book Search and Availability Check
- Borrow and Return Books
- Admin Dashboard
- Book Management System
- Lending History Tracking
- Responsive User Interface

---

## Technologies Used
- java
- springboot
- HTML
- CSS
- JavaScript
- MySQL
